🔹 Qu’est-ce que le HTML ?

HTML signifie HyperText Markup Language. C’est le squelette d’une page web. Exemple :

<h1>Bonjour le monde !</h1>

🔹 Et le CSS ?

CSS signifie Cascading Style Sheets. Il sert à styliser le HTML. Exemple :

h1 { color: red; }

🔹 Comment commencer ?

Tu peux coder en ligne avec CodePen ou JSFiddle, ou installer un éditeur comme VS Code.
